__Job Portal App__
- A fullstack Job Portal application built using the MERN stack (MongoDB, Express.js, React.js, Node.js). This platform connects job seekers and recruiters, enabling users to browse and apply for jobs, post new vacancies, and manage applications with a seamless user experience.

__Features__
- User Authentication: Secure login/register system for both job seekers and employers using JWT.​

__Profile Management:__
- Users can create and update profiles, upload resumes/photos, and add work experience, education, and skills.​

__Job Listings:__
-- View all job listings posted by employers with details like job title, company, description, salary, and location.​

__Advanced Search & Filters:__ 
- Search jobs by keyword, job type, experience level, location, and salary range.​

__Application Management:__
- Job seekers can apply and track applications. Recruiters can view and manage applicants.​

__Company Dashboard:__
-- Recruiters can create, update, and delete job postings and view candidate profiles.​

__Admin Panel:__
- Includes controls for managing users, job listings, and filtering companies/jobs.​

__Responsive Design:__
- Works across desktops, tablets, and mobile devices.​

__Tech Stack__
- Frontend: React.js, React Router, Redux Toolkit, Bootstrap/Tailwind CSS

- Backend: Node.js, Express.js
